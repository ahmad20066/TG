# Team Member Photos

## Instructions for Adding Team Photos

Place your team member photos in this directory with the following names:

1. **member1.jpg** - Mouhammad Al-Rifai (CEO & Founder)
2. **member2.jpg** - Abdelelah Bamarouf (Technical Director)
3. **member3.jpg** - Tarik Bamarouf (Creative Director)

### Photo Requirements:
- **Format**: JPG, PNG, or WebP
- **Dimensions**: Minimum 400x400px (square aspect ratio preferred)
- **Size**: Keep under 500KB for optimal web performance
- **Style**: Professional headshots with the red/black theme lighting work best

### Current Image Names Expected:
```
member1.jpg
member2.jpg
member3.jpg
```

### To Add More Team Members:
1. Add new photos to this folder
2. Update `src/pages/AboutPage/AboutPage.jsx`:
   - Import the new image at the top
   - Add a new team member object in the array (lines 358-376)
3. Include: name, role, motivation statement, and image reference

---

**Note**: The flip card animation will automatically work for all team members once their images are added!
